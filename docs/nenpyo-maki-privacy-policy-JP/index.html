    年表巻き プライバシーポリシー

    Posted on January 25, 2026

    Naoya Okayama（以下、開発者）は、年表巻きの運営に関し、個人情報の保護に関係する法令およびガイドラインを順守するとともに、ユーザーの皆さまからご提供いただいた情報を以下の通り取り扱います。なお、年表巻きのご利用を開始に伴い、このプライバシーポリシーに同意したものとします。

    1. 総則

    1.1. 本プライバシーポリシーの適用範囲

    本プライバシーポリシーは、年表巻きの全てのバージョンにて提供される全てのサービスに適用されます。開発者は別途、サービスごとのプライバシーポリシーまたは利用規約（以下「個別規約」）を定めることがあり、個別規約において別途情報の取り扱いを定める場合には、個別規約の定めが優先します。 なお、年表巻きに表示されるリンク等から外部のウェブサイト等に移動した場合のリンク先での情報の取り扱いについて、本プライバシーポリシーの適用はありません。

    1.2. 個人情報の定義

    本プライバシーポリシーにおいて、「個人情報」とは個人情報の保護に関する法律（平成十五年五月三十日法律第五十七号）第2条第1項により定義される個人情報を指します。

    2. ユーザーの個人情報の利用について

    2.1. 決済に必要な情報

    年表巻きの追加機能の購入など、一部の有料サービスでは決済のためにユーザーのクレジットカード情報等を利用することがありますが、これらの決済情報は第三者が運営する決済プラットフォームの側で保存されており、開発者は決済情報を取得しません。

    2.2. その他の情報

    開発者は上記にあげた情報以外の個人情報は利用しません。ご安心して年表巻きをお使いください。

    3. プライバシーポリシーの改定について

    本プライバシーポリシーを改定することがあります。改定した場合にはウェブサイトへの掲示その他ユーザーの皆様に分かりやすい方法で改定をお知らせします。改定後も引き続き毛糸を使用した場合、プライバシーポリシーの変更を受け入れたこととします。

    4. 開発者との連絡手段

    本プライバシーポリシーまたは年表巻きに関連するご質問、ご意見、その他のご連絡は開発者のX、または年表巻きのAppStoreレビューからお送りください。

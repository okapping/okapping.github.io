    アプリの特徴

    高い操作性

    操作のしやすさに重点を置いており、とてもシンプルなデザインになっています。
    スマートフォン操作が慣れない方でも安心してご利用いただけます。

    プライバシー

    お客様のプライバシーを第１に考えており、本アプリではお客様のデータは収集せず、また広告も表示致しません。
    全てオフラインで動作します。
